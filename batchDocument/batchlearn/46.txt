46、算数运算
【1】批处理中不存在浮点数，只有整数
【2】若除法的计算结果是浮点数，那么只输出整数部分
例如
@echo off
set /a a=9/10
echo %a%
显示结果为0
【3】如何进行浮点数的计算和显示呢
例1
@echo off
rem 计算40*123*0.01， 3*46*0.35
::0.01化1000,0.35化35000
set /a a=40*123*1000,b=3*46*35000
set a=%a:~,-5%.%a:~-5,2%
set b=%b:~,-5%.%b:~-5,2%
rem 为什么是-5呢，因为0.01到1000是扩大了100000倍
echo a=%a% b=%b%
pause
